# Требуется реализовать:

- Хочу контроллировать zIndex CursorPublicTracker из компонента Neprikosnovenna через props
- Хочу чтобы клики других пользователей также отслеживались на моем экране (хочу чтобы картинки курсоров синхронизировались)
- Хочу чтобы при клике на кнопку обезжирить у всех пользователей в сессии осищались оба Layer 1 и Layer 2
- Хочу чтобы курсоры других пользователей исчезали с экрана только тогда, когда пользователи покидают страницу / страница теряет фокус. 
- Хочу чтобы со всех устройств транслировались не только позиции курсоров, но и их текущие иконки (ссылки на них для производительности). При смене иконки други пользователи должны незамедлительно получать об этом сообщения и тоже изменять отображаемую иконки 
- Позиция локального курсора должна начинать транслдироваться только тогда, когда инициализируется targetRef.current (то есть когда курсор в первый раз появляется на экране)
